Schedule

Keep track of us as we work towards milestones and attend events!

2016

Date Milestone/Event Description
October 17, 2016 Project Plan Completed Version 1.0 of the project plan will be created and uploaded to the deliverables section of this website.
November 4, 2016 Software Requirements Specification Version 1.0 of the software requirements specification will be created and uploaded to the deliverables section of this website.
November 11, 2016 Project Iteration 1 Our first iteration will contain include an initial prototype of the user interface, database design, and a non-functional skeleton website. We will have begun to explore various scheduling algorithms.
December 9, 2016 Project Iteration 2 Our second iteration will include the completed database, frontend form submission pages, and cancellation pages. Both the student and volunteer submission backend integration will be completed. The skeleton website will be a functional website hereafter, integrated with backend functions. We shall use randomly generated data and user acceptance testing to thoroughly test our submission system at the completion of this iteration.
December 9, 2016 Design Document Version 1.0 of the design document will be created and uploaded to the deliverables section of this website.

Date Milestone/Event Description
